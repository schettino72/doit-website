If you use doit and think it is a useful project. Please consider to make a donation to support the developer/maintainer with maintanance tasks (bug fixes, merge patches, reply users) and further development.



InstallingÂ¶

  • Using pip:

    $ pip install doit

Latest version of doit supports only python 3. If you are using python 2:

$ pip install "doit<0.30"

  • Download the source and:

    $ pip install -e .

  • Get latest development version:

    $ git clone https://github.com/pydoit/doit.git

Note

  • doit depends on the packages pyinotify (for linux), macfsevents (mac).
