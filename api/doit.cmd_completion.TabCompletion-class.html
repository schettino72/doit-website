                      TabCompletion

generate scripts for tab-completion

If hardcode-tasks options is chosen it will get the task list from the current dodo file and include in the completion script. Otherwise the script will dynamically call `doit list` to get the list of tasks.

If it is completing a sub-task (contains ':' in the name), it will always call doit while evaluating the options.
